# xplore-rn
Xplore app in React Native

# Using
- yarn
- react-navigation
- react-native-fast-image

# Install
Install Yarn
```
npm install -g yarn
```

*on Windows, download yarn installer .msi*
<br><br>

Install all required dependencies
```
cd xplore-rn
yarn install
```
<br>

# Launch

Android:
```
react-native run-android
```

iOS:
```
react-native run-ios
```
